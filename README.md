# blog-vue3-master

vue3 + ts + vite2 搭建的完整博客项目，如果想查看基础架构，点击前往[基础项目](https://github.com/cqdcyonghao/blog-vue3/tree/basic)

### 一些配置上的变化

- ~~element plus~~ `naive-ui`

  naive-ui 风格更喜欢

- 新增 `tailwindcss`

  原子化 css 能使我的开发更节省时间，尤其是在页面布局略微复杂的时候

- ~~vuex~~ `pinia`

  pinia 简洁的多，不再需要花大量的时间去编写导入、调用函数以及嵌套 modules 等
